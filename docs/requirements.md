# 요구사항 정리

## 서비스 목적

소규모 개인사업자를 위한 AI 기반 간편장부 자동화 및 셀프 세무신고 준비 웹 서비스입니다. 초기 MVP는 AI 기능 없이 자료 정리, 장부 생성 준비, 리포트와 체크리스트 제공에 집중합니다.

모든 결과에는 아래 안내 문구를 포함합니다.

> 사용자 입력 및 업로드 자료 기준의 참고용 결과이며, 최종 신고 책임은 사용자에게 있음

## 법적 경계

- 세무신고 대행 서비스가 아닙니다.
- 사용자를 대신해 홈택스에 로그인하지 않습니다.
- 사용자를 대신해 신고서를 제출하지 않습니다.
- 세무대리, 신고대행, 절세 보장, 환급 보장과 같은 표현은 제품 문구에서 사용하지 않습니다.
- 제공 범위는 자료 정리, 간편장부 생성, 신고 준비 리포트, 홈택스 입력 가이드입니다.

## 기술 스택

### Frontend

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui 스타일 컴포넌트
- React Hook Form
- Zod
- TanStack Query

### Backend

- Spring Boot 3
- Java 21
- Spring Security
- JWT
- Spring Data JPA
- QueryDSL

### Database / Infra

- PostgreSQL
- Redis는 선택 사항
- 로컬 파일 저장소 우선, 운영에서는 AWS S3 또는 Cloudflare R2로 교체 가능하도록 추상화

### Excel / CSV

- Apache POI
- OpenCSV

### AI

- 초기에는 기능 구현 없이 `ai` 패키지 경계만 준비
- 추후 OpenAI API 또는 Gemini API 연동 가능하도록 설계

## MVP 기능

1. 회원가입/로그인
2. 사업자 정보 등록
3. 간편장부 대상 여부 예상 판정
4. 엑셀/CSV 파일 업로드
5. 거래 데이터 파싱
6. 거래 자동 분류
7. 거래 목록 조회/수정/삭제
8. 간편장부 자동 생성
9. 월별/연간 리포트 생성
10. 신고 준비 체크리스트 생성
11. 엑셀 다운로드
12. 대시보드

## 핵심 도메인

- `User`: 서비스 사용자
- `Business`: 사용자에게 속한 사업자 정보
- `UploadedFile`: 업로드 원본 파일 정보
- `Transaction`: 수입/비용 거래
- `CategoryRule`: 거래처명/키워드 기반 자동 분류 규칙
- `LedgerEntry`: 간편장부 행 데이터
- `TaxReport`: 월별/연간 요약 리포트
- `ChecklistItem`: 누락자료, 미분류거래, 증빙누락 등 신고 준비 체크 결과

## 간편장부 대상 예상 판정

- 신규 사업자는 간편장부 대상 가능성이 높다고 표시합니다.
- 직전연도 수입금액 기준:
  - A그룹: 농업, 임업, 어업, 광업, 도매 및 소매업 등은 3억원 미만
  - B그룹: 제조업, 숙박 및 음식점업, 건설업, 운수업, 정보통신업 등은 1억5천만원 미만
  - C그룹: 부동산임대업, 전문·과학 및 기술서비스업, 교육서비스업, 수리 및 기타 개인서비스업 등은 7천5백만원 미만
- 전문직 사업자는 수입금액과 관계없이 복식부기의무자로 표시합니다.
- 결과는 확정 판정이 아니라 입력값 기준 예상 판정으로 표시합니다.

## 보안 요구사항

- 비밀번호는 BCrypt로 해시합니다.
- JWT 인증을 사용합니다.
- 사업자별 데이터 접근 권한을 검증합니다.
- 파일 업로드 확장자와 크기를 제한합니다.
- 개인정보/사업자정보를 로그에 출력하지 않습니다.
- API 에러 응답에 민감정보를 포함하지 않습니다.
